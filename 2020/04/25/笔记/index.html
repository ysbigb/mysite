python符合多个条件

if i in range [a,b]

等价于

if i == a and i == b

字符属于某个区间

if "a" <= i <= "z"

第三个参数表示间隔几个数

range(a,b,c)

如(1-10)奇数可以用
range(1,10,2)

刷新输出

print("\r",end="")

end=”” 阻止换行， “\r”将光标移动到一开始的位置
